Collected Data / 収集した行動データ

  • Reason behind data selection / 選定した理由: 散歩は日常的に行う活動であり、のんびりと時間を気にせずゆったり歩くことを好む一方で、駅構内などでは移動効率を重視して速く歩くことが多い。同じ「歩く」という動作であっても、状況や感情によって動作パターンが大きく異なる点に興味を持った。本実験では、散歩（sanpo）、通学歩行（tuugaku_walk）、静止（still）、四足歩行（yotsunbai）の4種類の活動を選定した。散歩と通学歩行は同じ歩行動作でありながら速度やリズムが異なる。四足歩行はハイハイの動作であり、通常の歩行とは異なる動作パターンを示す。静止は動きがない状態の基準として選定した。これら4種類の活動は測定が容易であり、明確に区別できるため選定した。
  • Details about collection / 収集についての詳細

Features / 特徴量

  • Reason behind data selection / 選定した理由: 適切な特徴量を選定するため、GitHubなどで公開されている既存の行動認識データセット（UCI HAR Dataset）を用いて検証を行った。しかし、本実験で収集したデータは各活動2分間と比較的短いため、複雑な特徴量を使用すると過学習のリスクが高まることが判明した。そのため、時領域の基本的な統計特徴量（max, min, average, variance）に限定し、X, Y, Z軸それぞれについて計12個の特徴量を使用することとした。

ML Model / 機械学習モデル

  • Algorithm you used and the training result / 採用したアルゴリズムや学習結果の詳細: Pythonのscikit-learnライブラリを使用し、決定木（Decision Tree Classifier）アルゴリズムを採用した。Gini不純度を基準として、10-fold Cross-Validationで評価を行った結果、精度96.54%（標準偏差0.0355）を達成した。各活動の精度は、静止（still）98.7%、散歩（sanpo）98.7%、通学歩行（tuugaku_walk）93.7%、四足歩行（yotsunbai）94.9%であった。特徴量は12個（X, Y, Z軸それぞれのmax, min, average, variance）を使用した。
